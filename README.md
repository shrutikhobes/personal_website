shrutikhobes.com
=========

[View Site](https://shrutikhobes.com)

Building
--------

```bash
git clone git@github.com:shrutikhobes/shrutikhobes.com.git
cd shrutikhobes.com
npm install
npm run dev
# Open localhost:4003/index-dev.html in your browser
```

Building for Production
--------

```bash
npm run build
```
